# Supported Arrow data types

The following [Arrow data types](https://arrow.apache.org/docs/cpp/api/datatype.html) are supported by the current SDK version:

 - UINT8
 - UINT16
 - UINT32
 - UINT64
 - INT8
 - INT16
 - INT32
 - INT64
 - STRING
 - LIST
 - STRUCT
 - MAP
 - BOOL
 - FLOAT
 - DOUBLE
 - BINARY
 - DECIMAL128
 - DATE32
 - TIMESTAMP
 - TIME32
 - TIME64
 - NA
